## エラー時のユーザー体験

### リンクコード期限切れ

```
ユーザー: A1B2C3D4

ボット: ⚠️ このコードは有効期限が切れています。
        診断ページから新しいコードを取得してください。
        👉 診断ページへ
```

### 無効なコード入力

```
ユーザー: こんにちは

ボット: 💡 診断結果を連携するには、
        診断完了後に表示されるコードを
        入力してください。

        まだ診断を受けていない場合は
        👉 診断を始める
```
